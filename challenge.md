Challenge:
1. Create an EMPTY remote repository on GitHub.
2. Download the files from the scrim and put them in a local folder (.github won't be included. That's okay for now)
3. In that folder, initialize your local repo and connect and push it to your remote repo with these commands.
  1. git init
  2. git add .
  3. git commit -m "Initial commit"
  4. git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
  5. git branch -M main
  6. git push -u origin main
